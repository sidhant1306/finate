# Finate 💰
### Personal Finance & Investment Tracker

A full-stack financial management platform built with Spring Boot and React. Finate helps users track expenses, manage budgets, monitor stock portfolios with live prices, and get AI-generated weekly financial insights.

🌐 **Live Demo:** http://13.60.85.6/ &nbsp;|&nbsp; 📦 **Repo:** [github.com/sidhant1306/finate](https://github.com/sidhant1306/finate)

---

## ✨ Features

### 🔐 Authentication & Security
- JWT-based stateless authentication
- OTP-based email verification on signup
- BCrypt password hashing
- Role-based access control

### 💳 Wallet & Subscriptions
- Razorpay-integrated wallet with real money top-up
- Premium subscription via Razorpay (unlocks AI insights + stock features)
- Wallet balance management with transaction history

### 📊 Budget & Expense Management
- Monthly budget creation with category-wise limits
- Automatic budget reset via scheduled job (1st of every month)
- Income and expense tracking with category tagging
- Real-time budget utilization overview

### 📈 Stock Watchlist & Portfolio
- Add stocks to watchlist with live price broadcasting via **WebSocket**
- Portfolio tracking with live P&L and realized P&L
- Prices fetched from **Finnhub API** and broadcast to all connected clients in real time

### 🤖 AI-Powered Weekly Insights
- Weekly financial summary generated by **Spring AI + Gemini**
- Analyzes spending patterns, budget adherence, and portfolio performance
- Insights cached in **Redis** (TTL: 7 days) — regenerated every Monday via scheduler
- Async email delivery of weekly report to user's inbox

## 🚀 Running Locally

### Prerequisites
- Docker & Docker Compose
- Razorpay API keys
- Finnhub API key
- Google Gemini API key

### Steps

```bash
# Clone the repo
git clone https://github.com/sidhant1306/finate

# Set up environment variables
cp .env.example .env
# Fill in your API keys in .env

# Start all services
docker compose up --build
```

App will be available at `http://localhost`

### Environment Variables

```env
# Database
POSTGRES_USER=your_user
POSTGRES_PASSWORD=your_password
POSTGRES_DB=finate

# Redis
REDIS_HOST=redis
REDIS_PORT=6379

# JWT
JWT_SECRET=your_jwt_secret

# Razorpay
RAZORPAY_KEY_ID=your_key_id
RAZORPAY_KEY_SECRET=your_key_secret

# Finnhub
FINNHUB_API_KEY=your_finnhub_key

# Gemini
GEMINI_API_KEY=your_gemini_key

# Email (SMTP)
MAIL_USERNAME=your_email
MAIL_PASSWORD=your_app_password
```

## 🔑 Key Technical Decisions

- **BigDecimal** for all monetary values — avoids floating point precision errors
- **Unidirectional JPA relationships** — simpler, avoids accidental cascades
- **Redis caching** with `JacksonJsonRedisSerializer` + explicit type parameters for Spring Boot 3 / Jackson 3.x compatibility
- **`SecurityUtils` as a Spring `@Component`** — enables SpEL expressions in `@CacheEvict` annotations
- **Multi-stage Dockerfiles** — lean production images for both backend (JRE only) and frontend (Nginx)
- **WebSocket over polling** — real-time stock prices without hammering the Finnhub API
